# An empirical study of Fictitious Play for estimating Nash equilibria in first-price auctions with correlated values

This is the code for the version of the paper https://arxiv.org/pdf/2108.04506 that has been communicated at DynaFront https://sites.google.com/view/dynafrontneurips25, a Neurips2025 workshop. 

# Installation


## Install julia

`
yes '' | (curl -fsSL https://install.julialang.org | sh)
. ~/.bashrc
`

## Run the installer
`
julia installer.jl 
`

## Choose the number of thread
`
export JULIA_NUM_THREADS=16
`
